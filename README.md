# Terraform Azure Remote Backend

This project demonstrates how to provision Azure resources using Terraform and configure a remote backend using Azure Storage.

## 🚀 What I Did

* Created Resource Group
* Created Storage Account
* Created Blob Container
* Configured Remote Backend

## 🛠️ Tools

* Terraform
* Azure CLI
* Microsoft Azure

## 📌 Key Learning

* Infrastructure as Code (IaC)
* Remote state management
* Backend configuration in Terraform

## ▶️ How to Run

terraform init
terraform plan
terraform apply
